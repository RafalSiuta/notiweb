About app

Noti is an application project for managing notes and tasks in a calendar with an open-source code created in Flutter technology, which uses the Dart programming language. The main goal of the application is to offer the user to manage their records offline and synchronize these records with another device such as a desktop via Bluetooth without the need to connect to the Internet. The project is in a continuous development phase, in the future plans are to launch this application on other platforms such as IOS, MacOS, Linux and Windows.
